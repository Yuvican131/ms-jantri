# **App Name**: GridSheet Manager

## Core Features:

- Grid Sheet Display: Display a customizable 10x10 grid sheet 1 to 100 number box  with editable cells.
- Tabbed Navigation: Tabbed navigation for 'SHEET', 'CLIENTS', and 'ACCOUNTS'.
- Sheet Selection: Dropdown menu to select existing sheets or create new ones.
- Export to CSV: Functionality to export sheet as .csv for import into other accounting tools.
- Manage Clients: Add new entries for tracking individual Clients. There should also be the possibility to delete and update data about clients
- Manage Accounts: Ability to make notes on individual Accounts for enhanced auditing.
- AI Data Completion: Use generative AI to provide data validation and data completion recommendations. An LLM is used as a tool to validate cell contents based on user-defined rules and commonly used account names.

## Style Guidelines:

- Primary color: Violet (#A06CD5) to convey professionalism and trustworthiness.
- Background color: Light grey (#F0F0F0) for a clean, uncluttered interface.
- Accent color: Teal (#30D5C8) for interactive elements to ensure affordance and call attention to key functionalities.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, neutral look appropriate to a business application.
- Simple, geometric icons for navigation and actions.
- Clean and minimalistic layout to reduce visual clutter and enhance usability.
